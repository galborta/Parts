# Parts

**Talk to Your Inner Parts. Hear Them Talk Back.**

An IFS (Internal Family Systems) therapy companion powered by ElevenLabs voice AI and Cloudflare Workers. Built for ElevenHacks #2.

## Features

- Voice-based dialogue with different internal parts using ElevenLabs AI
- Multi-turn conversations with memory and insight tracking
- Real-time WebSocket communication via Cloudflare Durable Objects
- Google authentication
- Cloud-hosted on Cloudflare Workers + Pages

## Tech Stack

- **Frontend**: Next.js 15 + React + TypeScript + Tailwind CSS
- **Backend**: Cloudflare Workers + Durable Objects
- **Voice AI**: ElevenLabs API
- **Auth**: NextAuth.js (Auth.js) + Google OAuth
